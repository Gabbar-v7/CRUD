## Getting Started

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Gabbar-v7/CRUD.git
   ```

2. **Navigate to the Project Directory:**
   ```bash
   cd CRUD
   ```
4. **Get Dependencies:**
   ```bash
   flutter pub get
   ```
5. **Run the App:**
   ```bash
   flutter run
   ```
